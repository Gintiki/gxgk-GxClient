#中兴802.1x认证附带天翼web认证客户端
    	根据yxz广科校园网认证客户端源码进行二次开发，网页认证模式即将更改，所以停止更新，新版客户端雏形已完成，支持中兴+新天翼校园认证模式，考虑加入锐捷，但是没有环境测试
##编译环境
	VS2015
	所需库：curl、winpcap
##不足之处
	1.残留过多过时代码，代码累赘，函数大都使用窗口类
	2.MFC界面，界面开发难度大
